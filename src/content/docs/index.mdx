We're a vanilla-style Minecraft server, and this website is our handbook. It

## Using this handbook

This website documents everything you should need to know about Vanillacraft:
the vanilla-style Minecraft server.

If you have any other questions, please ask in our [discord server](
https://discord.gg/EFygPtvrvn)! But most information can be found in this book,
so please use the search feature (at the top) or the sidebar of contents to try
to find what you need first.
