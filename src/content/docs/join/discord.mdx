Having trouble joining our Discord server? Here's a step-by-step guide to 
installing Discord and joining the server.

## Step 1: Install Discord

If you don't have Discord installed, you can download it from the
[Discord website](https://discord.com/download), or your phone's app store if
you're on mobile.

You can also use the web version at https://discord.com/app, you still need
an account though.

## Step 2: Create an account

Create a Discord account, either in the app, or on the website. You just need an
email address and a username (our server's security settings require a verified
email address, so make sure to verify it!).

> Please note that you need to be 13 years or older to create a Discord account,
  so if you're younger than that, you won't be able to join our server. We don't
  enforce a strict age requirement on Vanillacraft, but you must be legally old
  enough to use Discord and Minecraft to join.

## Step 3: Join the server

Once you have a Discord account, you can join our server by clicking on this
invite link: https://discord.gg/EFygPtvrvn.

This will show a preview of the server, and you can click `Accept Invite` to
join and get talking to the community!

<LinkButton href="https://discord.gg/EFygPtvrvn">Join the Discord server</LinkButton>

## Next steps

Once you've joined the server, you can introduce yourself, and then start your
application!

Please see [How to join](/join/how) for more information on how to start
playing, and the application process.

If you have any questions, please feel free to make a question ticket (see
[Making a ticket](/ticket/create) if you're not sure how to do that).
