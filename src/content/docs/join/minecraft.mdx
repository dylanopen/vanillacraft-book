As soon as you have been accepted and [linked your account](/join/link), you
can join the Minecraft server and start playing!

Not sure how? See below.

## Steps to join

<Steps>

1. Open Minecraft.
2. Click "Multiplayer".
3. Click "Add Server".
4. In the "Server Name" field, enter "Vanillacraft" (or whatever you want!).
5. In the "Server Address" field, enter the server IP, which you can find in the
   `#🕹️-server-ip-🕹️` channel.
6. Click "Done" to add the server to your list.
7. Double click the server in your list to join and start playing!

</Steps>

## Next steps

Now that you're in, you can start exploring the server and meeting other
players.

If you'd like to be shown around, please see [Getting a tour](/join/tour) for
more information on how to get a tour of the server.
