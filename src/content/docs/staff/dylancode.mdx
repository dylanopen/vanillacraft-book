## About me

Hi, I'm dylancode! I like programming, Minecraft and other things.

I created Vanillacraft back in May of 2025, and I've run the server ever since.

I started the server because I wanted to create a fun, friendly, and
vanilla-style Minecraft server for me and some friends to play on, and it
quickly grew into a larger community of amazing players.

I don't play the server too often, but I work on various backend details (such
as keeping the server bug-free, managing plugins and configuration, and coding
custom, non gameplay-changing, plugins to add fun features and moderation tools
to the server).

I also help out with moderation and reviewing tickets from time to time.
